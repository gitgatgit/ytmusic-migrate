🎵 YTMusic Migrate

YouTube Music Account Migration Tool - Open source CLI tool for migrating your YouTube Music data between accounts.

✨ Features

  • Playlist Migration - Transfer all your custom playlists with videos
  • Liked Songs - Migrate your Liked Music and Liked Videos
  • Watch Later - Sync your Watch Later queue
  • Subscriptions - Copy all your channel subscriptions
  • Resume Support - Continues from where it left off if interrupted
  • Duplicate Detection - Skips already migrated items

📦 Installation

pip install google-auth-oauthlib google-api-python-client

Or download the script directly from GitHub.

🔑 Setup

  1. Create a project at Google Cloud Console
  2. Enable YouTube Data API v3
  3. Create OAuth credentials (OAuth client ID - Desktop app type)
  4. Save as credentials.json in the same folder as the script

🚀 Usage

# Migrate everything
python ytmusic_migrate.py --all

# Migrate only playlists
python ytmusic_migrate.py --playlists

# Scan what needs to be migrated (no changes)
python ytmusic_migrate.py --all --scan-only

# Reset migration state
python ytmusic_migrate.py --reset-state

⚡ YouTube API Quota

This tool uses the official YouTube Data API v3. Each operation consumes quota units:

  • Read operations (list): 1-3 units
  • Write operations (insert): 50 units

Default quota: 10,000 units per day

Example migration costs:

  • 200 subscriptions: 200 × 50 = 10,000 units
  • 1,000 liked songs: 1,000 × 50 = 50,000 units
  • 10 playlists with 100 videos each: 1,000 × 50 + 10 × 1 = 50,010 units

Note: Large migrations may require quota increase requests.
